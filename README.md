# WarehousePro Vercel Project

This is a ready-to-deploy Next.js project for your warehouse/logistics company website.

## Before you deploy
Open `app/page.tsx` and update:
- company name
- phone number
- service area
- contact text

## Run locally
```bash
npm install
npm run dev
```

## Upload to GitHub
1. Create a new repository in GitHub
2. Upload all files from this folder
3. In Vercel, click **Add New > Project**
4. Import your GitHub repository
5. Click **Deploy**
